The specific numeric caps are shown at the top of [y2pal.com/split-pdf](https://y2pal.com/split-pdf). They're adjusted periodically; always check the live page for today's caps.

## How the Free Tier Is Counted

- **Operations** = one split job (regardless of how many output files it produces).
- **File size** = the source PDF's size at upload, not the combined output size.
- **Daily count** = resets at UTC midnight.

A single 300 MB upload that produces 50 output files counts as **one operation** and requires an upload-size cap that covers 300 MB.

## What Happens When a Limit Is Reached

| Limit Hit                  | Behavior                                            |
|----------------------------|-----------------------------------------------------|
| File size too large        | Upload rejected with size message                   |
| Daily operation count used | New upload blocked until UTC midnight reset         |
| Concurrent session limit   | New tab/window sees "only one session at a time"    |

## Tips for Working Within Free-Tier Limits

1. Pre-split very large PDFs locally once to fit within the upload cap, then use Y2Pal for chapter-level splits.
2. Plan the day's work — 5 free operations is enough for typical ad-hoc use; larger batch work needs lifetime.
3. Use bookmark or every-N modes that produce many output files in a single operation; each counts as just one operation.

## When to Upgrade to Lifetime License

- You regularly upload PDFs above the free size cap.
- You process more than the daily operation count.
- You want multiple tabs/sessions active simultaneously.
- You want priority queue placement for faster processing on large files.

## How to Upgrade

Purchase from [y2pal.com/split-pdf](https://y2pal.com/split-pdf). The LFT (lifetime) option is a one-time payment, not a subscription. After payment, the license is tied to your purchase email and unlocks on any device where you sign in with that email.

Y2Pal does not require payment before using the free tier — you can evaluate the tool fully before buying. Refund terms are on the purchase page.
