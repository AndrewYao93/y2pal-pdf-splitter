# How to Split a PDF Online for Free

Splitting a PDF online sounds simple, but most tools either watermark the output, cap the file at 10 MB, or force an account sign-up. Y2Pal PDF Splitter runs entirely in the browser, produces clean unwatermarked output, and has no mandatory sign-up for the free tier.

## Why Split a PDF Online?

- **No installer** — the tool is a URL, not a desktop app. Works the same on Windows, macOS, Linux, and ChromeOS.
- **No admin rights needed** — ideal for work or school machines where software install is locked.
- **Cross-device** — start on a laptop, finish on a phone; the tool is identical.
- **No account** — the free tier processes the file and gives it back without asking for an email.

Files are removed from the processing service after the session ends — nothing is retained long-term.

## Step-by-Step Guide

### Step 1: Open the Tool

Visit [y2pal.com/split-pdf](https://y2pal.com/split-pdf). The drop zone is the first thing on the page.

### Step 2: Upload the PDF

Drag the source PDF onto the drop zone, or click **Upload File** and pick it from disk. Transfer uses HTTPS (TLS 1.3).

### Step 3: Choose How to Split

Three options:

- **By page range** — enter `1-10, 15, 20-25` style ranges.
- **By bookmark** — auto-detect the PDF's table of contents.
- **Every N pages** — divide into fixed-size chunks.

### Step 4: Process and Download

Click **Split PDF**. When processing finishes, download individual files or click **Download All** for a zip.

## Tips

1. Free-tier file-size caps are shown at the page header — check before uploading a very large file.
2. For PDFs over 500 MB, split once locally with a desktop tool to halve the size, then upload each half.
3. If you are splitting confidential documents, close the browser tab after downloading — this ends the session and triggers server-side cleanup.

## Requirements

- A modern browser: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+.
- Stable network with at least 2 Mbps upload for large files.
- No account required for the free tier.
