## Capture Guidelines

- Use a clean, empty browser window — no unrelated tabs, extensions visible, or bookmark bars.
- Blur or redact any filenames or page contents that could identify real documents.
- Use the English UI unless the repository is a localized variant.
- Capture at 100% browser zoom for predictable pixel dimensions.
- Prefer a light-mode screenshot for better contrast against GitHub's default theme.

## Placeholder Behavior

If these screenshots are missing, the tutorial section in `README.md` shows broken-image icons. Either supply real screenshots before publishing or remove the image references from `README.md`.
